후보자 관리란?

후보자 관리는 RCWS 시스템의 핵심 기능으로, 채용 후보자의 등록부터 최종 채용까지의 전체 프로세스를 관리합니다.

후보자 관리의 주요 기능
  • 후보자 정보 등록 및 관리
  • 이력서 및 자기소개서 업로드
  • 후보자 상태 관리 (대기/승인/거절)
  • 평가 및 면접 일정 관리
  • 채용 프로세스 추적
1. 후보자 등록하기
등록 방법
  1. 후보자 목록 페이지에서 "새 후보자 추가" 버튼 클릭
  2. 기본 정보 입력 (이름, 이메일, 전화번호)
  3. 지원 직무 및 경력 정보 입력
  4. 이력서 및 자기소개서 업로드
  5. 추가 정보 및 메모 입력
  6. "저장" 버튼 클릭
필수 입력 항목
  • 이름 - 후보자의 실명
  • 이메일 - 연락용 이메일 주소
  • 지원 직무 - 지원하는 직무명
  • 이력서 - PDF 형식의 이력서 파일
등록 시 주의사항
  • 이력서는 PDF 형식만 업로드 가능 (최대 10MB)
  • 프로필 이미지는 JPG, PNG 형식만 가능 (최대 5MB)
  • 이메일 주소는 중복 등록이 불가능합니다
  • 등록 후에는 수정이 가능하므로 정확히 입력해주세요
2. 후보자 검토 및 승인/거절
검토 프로세스
  1. 후보자 목록에서 검토할 후보자 선택
  2. 상세 정보 및 이력서 확인
  3. 자격 요건 및 적합성 검토
  4. 승인 또는 거절 결정
  5. 결정 사유 입력 (선택사항)
승인 시 진행사항
  • 후보자 상태가 "승인됨"으로 변경
  • 평가 단계로 자동 진행
  • 관련 채용 생성
  • 평가자에게 알림 발송
승인 기준
  • 지원 직무와 경력이 일치
  • 필수 자격 요건 충족
  • 이력서 내용이 적절함
  • 조직 문화와 부합
거절 사유
  • 자격 요건 미달
  • 경력 부족
  • 기술 스택 불일치
  • 조직 문화 부적합
  • 직무 부적합
3. 후보자 정보 관리
정보 수정
  1. 후보자 상세 페이지에서 "편집" 버튼 클릭
  2. 수정할 정보 변경
  3. 새로운 파일 업로드 (필요시)
  4. 변경사항 저장
상태 관리
  • 대기 - 등록 후 검토 대기
  • 승인됨 - 검토 후 승인
  • 거절됨 - 검토 후 거절
  • 평가 중 - 평가 진행 중
  • 채용됨 - 최종 채용
정보 관리 팁
  • 중요한 정보 변경 시 메모에 변경 사유를 기록하세요
  • 파일 업로드 시 기존 파일은 자동으로 교체됩니다
  • 상태 변경은 신중하게 결정하고 적절한 사유를 입력하세요
  • 정기적으로 후보자 정보를 업데이트하여 최신 상태를 유지하세요
4. 후보자 삭제
주의사항

후보자 삭제는 되돌릴 수 없는 작업입니다. 신중하게 결정해주세요.

삭제 방법
  1. 후보자 상세 페이지에서 "삭제" 버튼 클릭
  2. 삭제 확인 페이지에서 후보자 정보 재확인
  3. 삭제 사유 확인
  4. "삭제 확인" 버튼 클릭
삭제 시 주의사항
  • 삭제된 후보자는 복구할 수 없습니다
  • 관련된 모든 평가 데이터가 함께 삭제됩니다
  • 업로드된 파일들도 모두 삭제됩니다
  • 채용 참여 내역도 삭제됩니다
팁과 모범 사례
이력서 관리

이력서는 PDF 형식으로 변환하여 업로드하면 호환성이 좋습니다.

메모 활용

후보자별 메모를 활용하여 중요한 정보나 특이사항을 기록하세요.

검색 기능

후보자 목록의 검색 기능을 활용하여 특정 후보자를 빠르게 찾으세요.

필터링 활용

상태별, 직무별 필터링을 활용하여 후보자를 체계적으로 관리하세요.

빠른 링크
후보자 목록 새 후보자 등록 평가 관리 채용
상태별 가이드
대기 검토 대기 중인 후보자
승인됨 검토 후 승인된 후보자
거절됨 검토 후 거절된 후보자
평가 중 평가 진행 중인 후보자
채용됨 최종 채용된 후보자
문제 해결
파일 업로드가 안 돼요

파일 형식과 크기를 확인해주세요. (PDF: 10MB, 이미지: 5MB)

후보자를 찾을 수 없어요

검색 기능을 활용하거나 필터를 확인해보세요.

승인/거절 버튼이 안 보여요

후보자 상태가 "대기" 상태인지 확인해주세요.
