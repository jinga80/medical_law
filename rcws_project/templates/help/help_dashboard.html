대시보드란?

대시보드는 RCWS 시스템의 메인 페이지로, 채용 프로세스의 전체 현황을 한눈에 파악할 수 있는 핵심 화면입니다.

대시보드의 주요 기능
  • 실시간 채용 현황 통계
  • 최근 활동 내역
  • 빠른 액세스 메뉴
  • 시스템 알림
대시보드 구성 요소
통계 카드

상단에 위치한 4개의 카드로 주요 통계를 표시합니다.

  • 총 후보자 수
  • 진행 중인 채용
  • 완료된 채용
  • 평가 대기
최근 활동

시스템에서 발생한 최근 활동들을 시간순으로 표시합니다.

  • 새 후보자 등록
  • 평가 완료
  • 상태 변경
  • 사용자 로그인
알림 섹션

중요한 알림과 메시지를 표시합니다.

  • 평가 대기 알림
  • 시스템 공지사항
  • 업데이트 정보
  • 긴급 메시지
빠른 액세스

자주 사용하는 기능에 빠르게 접근할 수 있습니다.

  • 후보자 등록
  • 사용자 관리
  • 보고서 생성
  • 설정 변경
사용 방법
1단계: 로그인
  1. 브라우저에서 RCWS 시스템에 접속
  2. 사용자명과 비밀번호 입력
  3. 로그인 버튼 클릭
  4. 자동으로 대시보드로 이동
2단계: 현황 확인
  1. 상단 통계 카드 확인
  2. 최근 활동 내역 검토
  3. 알림 메시지 확인
  4. 전체적인 현황 파악
3단계: 작업 수행
  1. 빠른 액세스 버튼 활용
  2. 상단 메뉴를 통한 이동
  3. 필요한 기능 선택
  4. 작업 완료 후 대시보드로 복귀
4단계: 모니터링
  1. 정기적으로 대시보드 확인
  2. 새로운 알림 체크
  3. 통계 변화 추적
  4. 시스템 상태 모니터링
팁과 트릭
새로고침 활용

대시보드를 새로고침하면 최신 데이터를 확인할 수 있습니다.

즐겨찾기 설정

대시보드 URL을 브라우저 즐겨찾기에 추가하면 빠른 접근이 가능합니다.

알림 설정

브라우저 알림을 허용하면 중요한 업데이트를 놓치지 않을 수 있습니다.

검색 활용

상단 검색창을 활용하여 특정 후보자나 정보를 빠르게 찾을 수 있습니다.

빠른 링크

  • 후보자 관리
  • 사용자 관리

브라우저를 새로고침하거나 다시 로그인해보세요.

통계가 업데이트되지 않아요

페이지를 새로고침하거나 잠시 후 다시 확인해보세요.

브라우저 알림 권한을 확인하고 설정을 변경해보세요.
