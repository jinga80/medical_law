평가 관리란?

평가 관리는 후보자의 역량과 적합성을 객관적으로 평가하여 채용 결정을 지원하는 시스템입니다. 다양한 평가 항목과 기준을 통해 체계적인 평가를 진행할 수 있습니다.

평가 관리의 주요 기능
  • 평가 항목 및 기준 설정
  • 다중 평가자 평가 시스템
  • 평가 결과 통계 및 분석
  • 평가 히스토리 관리
  • 평가 피드백 시스템
평가 프로세스
1단계: 평가 대상 선정

승인된 후보자 중 평가 대상자를 선정합니다.

  • 후보자 자격 확인
  • 평가 일정 조율
  • 평가자 배정
2단계: 평가 진행

설정된 평가 기준에 따라 평가를 진행합니다.

  • 평가 항목별 점수
  • 평가 의견 작성
  • 평가 결과 저장
3단계: 결과 분석

평가 결과를 종합하여 분석합니다.

  • 종합 점수 계산
  • 평가자별 비교
  • 결과 리포트 생성
4단계: 최종 결정

평가 결과를 바탕으로 최종 결정을 내립니다.

  • 합격/불합격 결정
  • 피드백 전달
  • 다음 단계 안내
평가 항목
기술 역량 (40%)
  • 전문 기술 지식
  • 문제 해결 능력
  • 코딩/개발 능력
  • 기술 트렌드 이해
경험 및 실적 (30%)
  • 관련 업무 경험
  • 프로젝트 실적
  • 성과 지표
  • 포트폴리오
소프트 스킬 (20%)
  • 의사소통 능력
  • 팀워크
  • 리더십
  • 적응력
문화 적합성 (10%)
  • 조직 문화 부합
  • 가치관 일치
  • 성장 의지
  • 장기 계획
평가 방법
평가 생성
  1. 평가 목록에서 "새 평가" 버튼 클릭
  2. 평가 대상 후보자 선택
  3. 평가 항목 및 기준 설정
  4. 평가자 지정
  5. 평가 일정 설정
  6. 평가 생성 완료
평가 진행
  1. 평가 페이지 접속
  2. 각 항목별 점수 입력
  3. 평가 의견 작성
  4. 평가 결과 저장
  5. 평가 완료 확인
평가 시 주의사항
  • 객관적이고 공정한 평가를 위해 편견을 배제하세요
  • 구체적인 예시와 함께 평가 의견을 작성하세요
  • 평가 기준을 일관성 있게 적용하세요
  • 평가 결과는 후보자에게 적절한 피드백을 제공하세요
평가 결과 분석
분석 항목
  • 종합 점수 및 순위
  • 항목별 세부 분석
  • 평가자별 의견 비교
  • 강점 및 개선점
  • 합격 기준 대비 결과
결과 활용
  • 채용 결정 근거
  • 후보자 피드백
  • 평가 시스템 개선
  • 향후 채용 기준 수정
팁과 모범 사례
명확한 기준

평가 기준을 명확히 정의하고 모든 평가자가 동일하게 적용하세요.

다중 평가

여러 평가자의 의견을 종합하여 객관성을 높이세요.

기록 관리

평가 과정과 결과를 상세히 기록하여 추적 가능성을 확보하세요.

지속적 개선

평가 결과를 분석하여 평가 시스템을 지속적으로 개선하세요.
