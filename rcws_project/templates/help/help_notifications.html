알림이란?

알림 시스템은 RCWS에서 발생하는 중요한 이벤트와 업데이트를 사용자에게 실시간으로 전달합니다. 업무 효율성을 높이고 중요한 정보를 놓치지 않도록 도와줍니다.

알림의 주요 기능
  • 실시간 이벤트 알림
  • 작업 할당 및 마감일 알림
  • 시스템 업데이트 정보
  • 중요한 상태 변경 알림
  • 개인화된 알림 설정

알림 확인하기
확인 방법
  1. 상단 네비게이션의 알림 아이콘 클릭
  2. 드롭다운 메뉴에서 알림 목록 확인
  3. 읽지 않은 알림은 빨간 배지로 표시
  4. 알림 클릭하여 상세 내용 확인
  5. "모든 알림 보기"로 전체 목록 확인
알림 표시
  • 읽지 않음 - 빨간 배지로 표시
  • 읽음 - 일반 텍스트로 표시
  • 중요 - 특별한 아이콘으로 표시
  • 만료 - 회색으로 표시
알림 확인 팁
  • 정기적으로 알림을 확인하여 중요한 정보를 놓치지 마세요
  • 읽지 않은 알림은 우선적으로 확인하세요
  • 중요한 알림은 별도로 기록하거나 북마크하세요
  • 불필요한 알림은 삭제하여 정리를 유지하세요
