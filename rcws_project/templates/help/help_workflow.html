개요
채용 관리란?

채용 관리 시스템은 리버스클리닉과 케이지아웃소싱 간의 헤드헌팅 협업을 위한 종합적인 채용 프로세스 관리 도구입니다.

주요 기능
  • 채용 프로세스 생성: 새로운 채용 프로세스를 생성하고 관리
  • 단계별 관리: 채용 프로세스를 단계별로 세분화하여 관리
  • 진행 상황 추적: 실시간으로 채용 진행 상황을 모니터링
  • 담당자 지정: 각 단계별로 담당자를 지정하여 책임 분담
  • 일정 관리: 마감일 설정 및 지연 알림
  • 보고서 생성: 채용 현황 및 성과 분석 보고서
채용 생성
새 채용 생성하기
  1. 채용 목록 페이지에서 "새 채용 생성" 버튼 클릭
  2. 필수 정보 입력:
    • 채용 제목: 채용 프로세스를 식별할 수 있는 명확한 제목
    • 설명: 채용에 대한 상세한 설명
    • 마감일: 채용 완료 예정일
    • 우선순위: 긴급도에 따른 우선순위 설정
  3. "생성" 버튼 클릭하여 채용 프로세스 생성
팁: 채용 제목은 구체적이고 명확하게 작성하여 나중에 쉽게 식별할 수 있도록 하세요.
채용 관리
채용 정보 수정

생성된 채용의 정보를 언제든지 수정할 수 있습니다.

  • 채용 상세 페이지에서 "수정" 버튼 클릭
  • 변경하고자 하는 정보 수정
  • "수정" 버튼 클릭하여 변경사항 저장
채용 삭제

더 이상 필요하지 않은 채용은 삭제할 수 있습니다.

  • 채용 상세 페이지에서 "삭제" 버튼 클릭
  • 삭제 확인 대화상자에서 "삭제 확인" 클릭
  • 주의: 삭제된 채용은 복구할 수 없습니다.
단계 관리
기본 채용 단계

새로운 채용을 생성하면 다음과 같은 기본 단계가 자동으로 생성됩니다:

  1. 채용 요청: 채용 요청서 작성 및 제출
  2. 구인 공고: 채용 공고 작성 및 게시
  3. 지원자 모집: 후보자 지원 및 모집
  4. 서류 심사: 이력서 및 자기소개서 검토
  5. 면접 일정 조율: 면접 일정 및 장소 조율
  6. 면접 진행: 실제 면접 실시
  7. 면접 평가: 면접 결과 평가 및 점수
  8. 최종 결정: 합격자 선정 및 결정
  9. 채용 확정: 최종 채용 확정 및 계약
단계별 작업
  • 시작하기: 단계를 시작하여 진행 상태로 변경
  • 완료하기: 단계를 완료하여 다음 단계로 진행
  • 되돌리기: 이전 상태로 되돌리기 (필요시)
  • 보류하기: 일시적으로 단계를 보류 상태로 변경
상태 관리
채용 상태
  • 대기중 - 채용이 생성되었지만 아직 시작되지 않음
  • 진행중 - 채용이 활발히 진행 중
  • 완료 - 채용이 성공적으로 완료됨
  • 취소 - 채용이 취소됨
  • 보류 - 일시적으로 중단됨
  • 검토중 - 검토 단계
  • 지연 - 마감일을 초과함
단계 상태

각 단계는 다음과 같은 상태를 가질 수 있습니다:

  • 대기중: 단계가 생성되었지만 아직 시작되지 않음
  • 진행중: 단계가 현재 진행 중
  • 완료: 단계가 성공적으로 완료됨
  • 보류: 일시적으로 중단됨
  • 취소: 단계가 취소됨
보고서
채용 현황 보고서

채용 진행 상황을 한눈에 파악할 수 있는 다양한 보고서를 제공합니다.

  • 진행률 차트: 전체 채용 진행률을 시각적으로 표시
  • 단계별 현황: 각 단계별 완료 현황
  • 담당자별 성과: 담당자별 작업 완료 현황
  • 지연 현황: 마감일을 초과한 채용 목록
보고서 생성
  1. 채용 상세 페이지에서 "보고서" 탭 클릭
  2. 원하는 보고서 유형 선택
  3. 필터 조건 설정 (기간, 담당자, 상태 등)
  4. "생성" 버튼 클릭하여 보고서 생성
  5. PDF 또는 Excel 형식으로 다운로드 가능
팁과 트릭
효율적인 채용 관리
채용 생성 시
  • 명확하고 구체적인 제목 사용
  • 상세한 설명으로 목표와 요구사항 명시
  • 현실적인 마감일 설정
  • 적절한 우선순위 지정
진행 중
  • 정기적으로 진행 상황 업데이트
  • 지연이 예상되면 즉시 담당자에게 알림
  • 중요한 결정사항은 메모에 기록
  • 관련 문서는 첨부하여 관리
주의사항:
  • 채용 삭제 시 모든 관련 데이터가 영구적으로 삭제됩니다.
  • 상태 변경은 신중하게 진행하세요.
  • 중요한 정보는 정기적으로 백업하세요.
