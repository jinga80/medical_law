보고서란?

보고서는 채용 프로세스의 현황과 성과를 데이터 기반으로 분석하여 제공하는 기능입니다. 의사결정을 지원하고 채용 전략을 수립하는 데 도움을 줍니다.

보고서의 주요 기능
  • 채용 현황 통계
  • 후보자 분석
  • 평가 결과 분석
  • 성과 지표 추적
  • 트렌드 분석
보고서 종류
채용 현황 보고서

전체 채용 프로세스의 현황을 한눈에 파악할 수 있습니다.

  • 후보자 등록 현황
  • 진행 단계별 통계
  • 승인/거절 비율
  • 채용 완료 현황
후보자 분석 보고서

후보자들의 특성과 패턴을 분석합니다.

  • 지원 직무별 분포
  • 경력 수준 분석
  • 지역별 분포
  • 평가 점수 분석
평가 성과 보고서

평가 과정과 결과를 분석합니다.

  • 평가자별 성과
  • 평가 항목별 분석
  • 평가 시간 분석
  • 평가 품질 지표
트렌드 분석 보고서

시간에 따른 변화와 트렌드를 분석합니다.

  • 월별/분기별 추이
  • 시장 변화 분석
  • 예측 모델링
  • 비교 분석
보고서 생성하기
생성 방법
  1. 보고서 목록에서 "새 보고서" 버튼 클릭
  2. 보고서 유형 선택
  3. 분석 기간 설정
  4. 필터 조건 설정
  5. 보고서 제목 입력
  6. "생성" 버튼 클릭
필터 옵션
  • 기간 - 분석할 기간 설정
  • 직무 - 특정 직무 필터링
  • 평가자 - 특정 평가자 필터링
  • 상태 - 후보자 상태별 필터링
보고서 생성 팁
  • 분석 목적에 맞는 적절한 기간을 설정하세요
  • 필요한 필터를 적용하여 정확한 데이터를 확보하세요
  • 보고서 제목은 내용을 명확히 표현하세요
  • 정기적으로 동일한 조건의 보고서를 생성하여 비교하세요
보고서 해석하기
주요 지표
  • 채용 전환율 - 후보자 등록 대비 채용 완료 비율
  • 평균 평가 점수 - 전체 후보자의 평균 평가 점수
  • 평가 완료율 - 평가 대상 대비 완료 비율
  • 평균 처리 시간 - 단계별 평균 처리 시간
분석 포인트
  • 트렌드 변화 패턴
  • 성과 개선 영역
  • 비효율적인 프로세스
  • 예상 문제점
해석 시 주의사항
  • 데이터의 맥락을 고려하여 해석하세요
  • 단순한 수치가 아닌 의미를 파악하세요
  • 여러 지표를 종합적으로 분석하세요
  • 변화의 원인을 추적하여 개선점을 도출하세요

보안 주의사항
  • 민감한 정보가 포함된 보고서는 신중하게 공유하세요
  • 공유 대상자의 권한을 확인하세요
  • 보고서 접근 로그를 정기적으로 확인하세요
  • 불필요한 보고서는 적절히 보관하거나 삭제하세요
