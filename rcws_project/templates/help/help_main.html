RCWS 사용 설명서에 오신 것을 환영합니다!

이 가이드를 통해 RCWS(Recruitment Candidate Workflow System)의 모든 기능을 쉽게 사용하실 수 있습니다.

빠른 시작 가이드
1. 대시보드 확인

시스템 전체 현황을 한눈에 확인하세요.

2. 후보자 등록

새로운 후보자를 시스템에 등록하세요.

3. 후보자 검토

등록된 후보자를 검토하고 승인/거절하세요.

4. 보고서 확인

채용 현황과 통계를 확인하세요.
